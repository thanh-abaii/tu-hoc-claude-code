# Danh sach Kiem tra Code Review

## Danh sach Kiem tra Bao mat
- [ ] Khong co thong tin dang nhap hoac bi mat duoc hardcode
- [ ] Xac thuc dau vao tren tat ca dau vao nguoi dung
- [ ] Phong ngua SQL injection (truy van tham so hoa)
- [ ] Bao ve CSRF tren cac thao tac thay doi trang thai
- [ ] Phong ngua XSS voi escape dung cach
- [ ] Kiem tra xac thuc tren endpoint duoc bao ve
- [ ] Kiem tra uy quyen tren tai nguyen
- [ ] Ma hoa mat khau an toan (bcrypt, argon2)
- [ ] Khong co du lieu nhay cam trong nhat ky
- [ ] Ap dung bat buoc HTTPS

## Danh sach Kiem tra Hieu suat
- [ ] Khong co truy van N+1
- [ ] Su dung index phu hop
- [ ] Cache duoc trien khai noi co loi
- [ ] Khong co thao tac chan tren luong chinh
- [ ] Async/await duoc dung dung
- [ ] Tap du lieu lon duoc phan trang
- [ ] Ket noi co so du lieu su dung pool
- [ ] Bieu thuc chinh quy duoc toi uu
- [ ] Khong tao doi tuong khong can thiet
- [ ] Phong ngua ro ri bo nho

## Danh sach Kiem tra Chat luong
- [ ] Ham < 50 dong
- [ ] Dat ten bien ro rang
- [ ] Khong co ma trung lap
- [ ] Xu ly loi dung cach
- [ ] Comment giai thich TAI SAO, khong phai LA GI
- [ ] Khong co console.log trong production
- [ ] Kiem tra kieu (TypeScript/JSDoc)
- [ ] Tuan thu nguyen tac SOLID
- [ ] Ap dung mau thiet ke dung cach
- [ ] Ma tu tai lieu

## Danh sach Kiem tra Kiem thu
- [ ] Unit test da duoc viet
- [ ] Bao quat cac tinh huong bien
- [ ] Kich ban loi da duoc kiem tra
- [ ] Integration test co mat
- [ ] Do phu > 80%
- [ ] Khong co test khong on dinh
- [ ] Mock phu thuoc ben ngoai
- [ ] Ten test ro rang
