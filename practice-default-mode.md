# Demo Chế Độ Mặc Định

File này được tạo ở **chế độ mặc định**.
Trong chế độ này, Claude yêu cầu xác nhận trước khi:
- Viết/chỉnh sửa file
- Chạy lệnh Bash
- Bất kỳ thao tác phá hủy nào

## Quy Trình Cấp Phép

1. Claude đề xuất hành động
2. Người dùng xem xét và chấp thuận/từ chối
3. Chỉ khi đó Claude mới tiếp tục

## Phù Hợp Cho
- Phát triển tương tác
- Học Claude Code
- Thay đổi mã nhạy cảm
- Kịch bản pair programming

## Chế Độ AcceptEdits
Trong chế độ acceptEdits, chỉnh sửa file được **tự động chấp thuận**.
Chỉ lệnh Bash cần xác nhận.
Phù hợp cho:
- Tái cấu trúc hàng loạt
- Sửa lỗi linting
- Tự động định dạng mã
